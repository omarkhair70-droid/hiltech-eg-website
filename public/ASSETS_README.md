# HILTECH Public Asset Guide

## Expected brand filenames
- `logo.png`
- `logo-dark.png`
- `favicon.ico`
- `favicon.png`
- `apple-touch-icon.png`
- `og-image.png`

## Optional folders
- `products/` (optional product images)
- `projects/` (optional real project photos)

## Recommended dimensions and formats
- `logo.png`: transparent PNG (SVG is also acceptable when supported by implementation).
- `favicon.ico` or `favicon.png`: 32x32 or 48x48.
- `apple-touch-icon.png`: 180x180.
- `og-image.png`: 1200x630.
- product images: use a consistent aspect ratio and export optimized files.

## Notes
- Keep files compressed and web-optimized to preserve fast loading.
- Use only approved, rights-cleared real assets.
